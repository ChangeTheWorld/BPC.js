# BPC.js
A JavaScript library used to generate 68.5KHz BPC which used for Radio controlled timepieces

```html
<script src="https://unpkg.com/bpc.js@0.0.2/bpc.js" type="text/javascript"></script>

<script type="text/javascript">
BPCStart();
</script>
```
